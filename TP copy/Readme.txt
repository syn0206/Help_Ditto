Readme*Project Name:Help, Ditto! *Project Description:Help, Ditto! is a game motivated by pokemon game series on nintendo. Ditto is a pokemon that can transform into any other pokemon. The player has to transform his/her ditto into another pokemon that is randomly put on the maze and get help from that transformed ditto to fight against the homework assignment. If the transformed ditto successfully attacks the assignment, the player wins the game.*How to run the project:Click on the Yuna Shin TP.py and run the code.*Libraries that need to be installed:None*Shortcut Commands:None
